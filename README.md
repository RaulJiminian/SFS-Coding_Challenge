# Strategic Financial Solutions Coding Challenge

## Coding Requirements

* [x] Fetch JSON data from Strategic Financial Solutions' API
* [x] Display JSON in table format
* [x] Add ADD / DELETE buttons (add generates a new account / delete removes account entry)
* [x] Add check boxes as the first column to check and uncheck each row in the table
* [x] Calculate and display the number of rows that have been checked
* [x] Calculate and display the Balance which is the sum of all the rows that have been checked
